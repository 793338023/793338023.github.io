还有就是最好把 nodemodules 里的所有包都重新安装一次，因为可能我们的配置并没有生效，这是因为 babel-loader 没有检测到 package.json 中的改变，一个最简单的解决方式就是直接删除掉 node_modules/.cache 文件夹，重新编译。

这样我们就可以兼容 ie11 的版本了。

当然，如果 ie11 还报错,可以试一下把babel-polyfill也安装上。
